# Boss直聘 数据采集与自动沟通工具

基于 DrissionPage 浏览器自动化的 Boss直聘（zhipin.com）岗位信息采集和批量打招呼工具。

## 功能

| 脚本 | 用途 |
|------|------|
| `claw.py` | 搜索岗位列表 → 采集详情（职位描述），输出 CSV |
| `auto_greek.py` | 搜索岗位列表 → 批量发送沟通请求（打招呼） |

### 特性

- **自动登录检测**：首次运行弹出浏览器登录，后续 cookie 持久化自动复用
- **断点续传**：中断后重跑自动跳过已处理的岗位
- **限流保护**：检测到请求频繁自动冷却重试，登录过期自动等待重新登录
- **失败重试**：每请求最多 3 次重试，递增等待

## 安装

```bash
pip install pandas drissionpage
```

## 使用

### 采集岗位数据

```bash
python claw.py
```

配置项（文件顶部）：城市代码 `CITY_CODE`、关键词 `KEYWORD`、最大翻页 `MAX_PAGES`

输出 `boss_jobs.csv`，包含岗位名称、薪资、技能要求、经验学历、公司信息、职位描述等。

### 批量打招呼

```bash
python auto_greek.py
```

输出 `greeted.txt`，格式：`岗位名称|公司名称|状态`

## 依赖

- Python 3.7+
- pandas — 数据处理
- DrissionPage — 浏览器自动化（会自动打开 Chrome/Edge）
